# Project and Task Management System

Web-based application for managing projects, tasks, employees, roles, and access rights in an IT service company.

## Tech Stack
- Frontend: React
- Backend: Spring Boot
- Database: PostgreSQL
- Security: Spring Security
- Persistence: JPA / Hibernate

## Project Structure
- `frontend/` React frontend
- `backend/` Spring Boot backend
- `docs/` UML, ER diagrams, screenshots, report assets
